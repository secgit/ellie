Create a smartphone game for kids in pure HTML/CSS/Javascript. The game features a princess wandering around a castle collecting items. The princess is controlled using gyroscopic controls. Each time the princess collides with an item, the player is asked to spell the item, if spelled correctly the princess obtains the item. When the item is successfully obtained a special image should briefly display showing the princess happily interacting with that item. There are ten unique rooms in the castle, that each contain one item. The princess navigates between the ten unique rooms using one or multiple doors in each room, when the princess collides with a door they move to the next room. Generate all the image assets necessary for the game.
